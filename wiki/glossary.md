# Glossary

- **SSE**: Server-Sent Events. The protocol used to stream text generation chunks back to the UI in real time.
- **Thinking Mode**: A configuration (specifically `thinkingLevel: 'HIGH'`) for supported models that allows them to produce multi-step reasoning before answering.
- **Gemma 4 31B Dense**: `gemma-4-31b-it`, the standard model with all parameters active.
- **Gemma 4 26B A4B (MoE)**: `gemma-4-26b-a4b-it`, a Mixture of Experts model with only a subset of parameters active per token.
